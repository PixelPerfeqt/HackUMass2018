# HackUMass2018 - Panic Button

Ever felt stressed while looking at your computer? Ever felt like the world was spinning out of control? Ever suspected that the world had an error, and it might be between keyboard and chair?

Then this is the program for you! Boot it up, click the button, and feel your heart be soothed.

# How To Use

Upon running the program, you will be greeted with a window with an empty text box and a button, like this:
<img src="https://user-images.githubusercontent.com/43099607/46910043-5441fa00-cf0b-11e8-85da-39ffe6419c38.png" width="90%"></img>

Clicking the button results in an affirmation being displayed:
<img src="https://user-images.githubusercontent.com/43099607/46910045-56a45400-cf0b-11e8-84e0-b600858ee033.png" width="90%"></img> 

Clicking the button again will cause a different affirmation to be displayed:
<img src="https://user-images.githubusercontent.com/43099607/46910047-5b690800-cf0b-11e8-95a3-900df39946af.png" width="90%"></img> 

The user can press the button as many times as they wish, until they feel calm and ready to proceed with life.
